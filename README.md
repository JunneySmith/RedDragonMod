=================================文化重制=================================
【政体设定】

为体现中央官制和地方制度在稳定性、可用性上的区别，天子和封臣将使用不同的政体。
新“天朝制”政体，具有原版所有的政体惩罚，由宋朝太宗朝及以后的皇帝持有。
原“天朝制”重命名为“贤能制”，取消政体惩罚，由霸权的封臣角色持有。
原“贤能制”重命名为“品级制”，数值不变。华夏文化的“品级制”角色可以花费50威望转为新的贤能制。
加入国库分成，朝廷地方分税、新“贤能制”和其他政体天子之间的联动机制，可以当标准的封建皇帝、选贤任能的理想中式君主和“众正盈朝”的假皇帝。
两个新的用来调节国库和税收政策的法律，“分税政策”和“国库政策”，用以平衡原版严重坐牢的经济数值。
为帝国级贤能制封臣增加了“并肩王”封臣契约，可以使用皇帝仪仗，可以更改自身及下属封臣的国库法律。

【玩法调整】

新增“投效”角色互动，贤能制角色在处于无头衔乡绅状态时可以选择成为国内另一官员或天子的封臣。
非“天朝制”天子将可以自由撤销朝贡，并不再消耗威望。
取士原则中以科举为独尊时将无法举办武举会试。
朝贡机制优化，法律允许时，天子的直属封臣也可以进行朝贡。天子新增了朝贡法律，可以选择完全禁止朝贡，或单独允许内藩封臣、外藩朝贡国觐见。
会试活动可以由帝国级的独立角色或有并肩王特权的总督举办。优化了一些项目的文本。整合中华决议将把头衔重命名为王畿。
新的法律组，可以限制贤能制总督的头衔上限并按品级给予加成。
会试新增了两个选项“考试资格”、“额外锁厅加试”，可以选择是否允许出身卑微者和在职官员参加科举。
出身卑微者通过会试将建立家族，开锁厅试的则在职官员参考者会有额外100分的加分。
新的宫廷职位“近臣”，成为天子或并肩王的门生能提高对应角色3级的贤能上限。

【王朝兴衰】

新的朋党“群雄”，时代对天子对外战争的限制与新的“天朝制”政体绑定。
对天子的特殊时代惩罚移入新的法律组“取士倾向”，启用科举取士倾向法律时会随时代切换自动切换。
获得天命将不再直接变更政体，而是触发事件自由决定，或是保留现有政体，或是从“郡国制”、“军镇制”、“贤能制”、“天朝制”中择一选择。
王朝兴衰机制更新，现在霸权在进入乱世后有两种发展：
如果天子是天朝制，则大体接近原版的炸裂模式，但只会影响中华本土的流官势力。
如果天子是贤能制，则销毁霸权头衔，主头衔外的封臣脱离，强大的法理内封臣脱离。

【其他内容】

1. 新文化兵士，军府重步兵、龙武军轻骑兵、神武军重骑三个文化特色兵种。
军府重步克制重骑、枪兵、象兵，龙武军轻骑克制弓兵、散兵、弓骑兵和游牧部众，神武军重骑克制重步、轻骑、弓骑兵、游牧部众。
三种特色兵种除弓骑、游牧部众以外的克制点数统一调整为1点。
2. 新文化公国建筑，折冲府，提供重步兵、重骑兵战力加成。官学、要塞城市，经济加成为主。
城市和公国建筑是通用建筑，有汉文化财政总管即可建设。
3. 新头衔，重建帝国级头衔西域都护府，永镇河西废除西夏重建凉国。凉国将成为河西走廊的法理王国，归义军将接手安西故地，同时归义军成为西域都护府法理头衔。
4. 新CB，两个宣战理由，可以对部落制国家无限使用。获取专属革新后可以摧毁部落制、游牧制持有的帝国级头衔，并将其法理整合入中华霸权。
5. 为文化领袖加入了创建归义军头衔决议，如果归义军头衔不存在，则文化领袖可以通过决议进行创建。
6..大部分兵士和军事惩罚移入士大夫传统，该传统将在宋朝开始的游戏中自动添加并替代汉唐遗风传统。
7. 新的专属文化传统。汉唐遗风，提供军事加成和社会倾向。农业专家，启用所有农耕类传统的建筑加成，提升洪泛平原和农田地形额外10%的发展度。
8. 新的专属文化革新，中华军制、华夏传承，启用文化特色兵种、宣战理由、并能够提前一个时代解锁长子继承制。

=================================政体=================================

【郡国并行制政体】

属于封建制的变体政体，有国库、贤能品级、可以控制流官、调整税法，豪族和东亚家宅住所。
可以自由选择分封封建制、贤能行政制、军镇制封臣，减少来自行政制和品级制封臣50%的税收和征召兵。

【军镇制】

封建制变体，归义军开局自带，在天朝体系下算作武职军官，可征调。可以使用牧群，从领地获取牧群的效率为游牧制政体的50%，从封臣和朝贡获取的牧群效率正常。内战启用依照忠顺度召集军头封臣加入战争的机制。
减少25的封臣上限，减少1点直辖上限，可以直辖城堡、牧场、部落。
“牙帐”机制，6孔位，可以使用营地建筑，受营地情绪影响，军镇制法律可解锁全部营地建筑。
“定居移民”内阁任务在军镇制下加强，转化游牧制土地时速度提升，且完成时将领地类型转化为城堡。
归义军开局时将默认启用军户选举制，军镇制角色将获得一支“义军”事件军队。
